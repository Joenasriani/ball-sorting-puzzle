BALL SORT 3D

A relaxing 3D color sorting puzzle game built for players who enjoy satisfying brain games, clean logic puzzles, smooth browser gameplay, and colorful casual challenges.

Move the colored balls between tubes, match every color, and solve each level using careful planning. The rules are simple: you can only move the top ball into an empty tube or onto another ball of the same color. Every puzzle is designed to be solvable, but each level asks you to think ahead, avoid blocking yourself, and use Undo wisely.

BALL SORT 3D is easy to learn, smooth to play, and perfect for quick puzzle sessions on desktop, laptop, tablet, or mobile.

HOW TO PLAY

1. Select a tube to pick up the top ball.
2. Select another tube to move the ball there.
3. You can move a ball only if the target tube is empty or has the same color on top.
4. Sort all balls so each tube contains only one matching color.
5. Use Undo if you make a mistake.
6. Complete the level and move to the next challenge.
